# Consequences

A gamified financial literacy application for secondary school students, built in Unity as a final-year MEng project at UCL.

## About

Consequences teaches young people core financial skills through interactive minigames. The app uses a text-to-speech model to read quiz questions aloud and guide players through gameplay, making it accessible to a wide range of learners.

The project is developed in collaboration with the **National Autistic Society** and **Motion Input Games**, supervised by Prof. Dean Mohamedally at UCL. It was presented at the **UCL AI Festival Showcase** to an audience of industry professionals and academics.

## Minigames

- **Budget & Spending** - Players make spending decisions and see the consequences of their choices over time.
- **Emergency Fund Building** - Players learn to set aside savings and handle unexpected expenses.

More minigames are planned.

## Tech Stack

- **Engine:** Unity (C#)
- **AI:** Text-to-speech model for narration and in-game guidance; re-engagement model planned to detect disengagement and prompt players back

## Status

This is an active final-year project (UCL MEng Mathematical Computation, 2025–2026). Features are still being developed and refined.
